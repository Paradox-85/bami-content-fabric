# Quarantine Directory

This directory contains **metadata records** for SVG assets requiring human visual review.

**Important:**
- Quarantine is a metadata/review queue, not proof of visual class or correctness.
- Assets listed in `review-required.yaml` must NOT be referenced as verified visual references in runtime code.
- No runtime code depends on this directory.
- `review-required` status means: "needs human visual inspection before classification can be considered reliable."
